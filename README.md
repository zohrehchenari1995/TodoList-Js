# 📝 Todo List Project – Confetti Animation, Filters, and Progress Bar

This is a **JavaScript-based Todo List application** with interactive UI features and smooth animations.

---

## 🚀 Live Demo

👉 **https://zohrehchenari1995.github.io/TodoList-Js/**

---

## ✨ Features

- ➕ Add new tasks  
- ❌ Delete tasks  
- ✔️ Mark tasks as completed  
- 🎉 Confetti animation when a task is completed  
- 🔄 Real-time progress bar updates  
- 🟢 Circular counter showing number of completed tasks  
- 🔍 Filter tasks by:
  - All
  - Completed
  - Incomplete
- 📉 Progress bar resets when tasks are unchecked or deleted  
- 📱 Responsive design

---

## 📊 How the Progress Bar Works

- Every time you **check**, **uncheck**, or **delete** a task:
  - The progress percentage is recalculated  
  - The circular counter updates  
  - If no completed tasks remain, the progress bar resets to zero
